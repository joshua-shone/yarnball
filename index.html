spacebar New unnamed node to right of cursor
enter↲ New unnamed node downward
(start typing alphanumeric) Overwrites name of node under cursor, or creates node
del Delete node at cursor
ctrl + ( ← → ↑ ↓ ) Move selected nodes
shift + ( ← → ↑ ↓ ) Drag selection box
ctrl + A Select all
ctrl + click (on node) Select all instances in connected group
ctrl + double click (on node) Select all instances
double click Select nearest group of connected nodes
ctrl + C Copy selected nodes/links
ctrl + V Paste nodes/links
ctrl + F Show find panel
tab Toggle link creation mode (and applies current node under cursor to new link)
ctrl + S Save to local storage
ctrl + shift + S Save to file (download)
(middle mouse button drag) Scroll/pan view
ctrl + Z Undo
ctrl + shift + Z Redo
